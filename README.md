# library
This project is about creating, a library for books, where the user can add the book he/she wants to read.
If  the book was read, it can be checked to indicate that.
Users are also able to delete the each book card if they like to.
This project has taught me to work with objects by utilizing constructions so I don't write same code again and again.
The function with that constructors can perform what it is intended to do as long as is being called.
Whenever is called it does the work.
